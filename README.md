# Prabhjot Kaur — Portfolio

Live site: https://prabhjot-kaurr.github.io

My personal portfolio as an AI Search Visibility Strategist — built to showcase
real SEO and GEO (Generative Engine Optimization) work, not just list skills.

## What's on the site

- **Work** — signature projects with a problem → action → result structure,
  including an AI citation-testing framework across ChatGPT, Gemini, Claude,
  and Perplexity, plus published blog links
- **Experience** — internship and work history timeline
- **Certifications** — SEO, GEO, and analytics credentials
- **Tools & Skills** — organized by category (GEO, SEO, content, outreach, data)
- **About** — background, education, and contact

## Built with

- Vanilla HTML, CSS, and JavaScript — no framework or site builder
- Google Fonts: Fraunces, Inter, IBM Plex Mono
- JSON-LD structured data (Schema.org Person) for search and AI-answer visibility
- Open Graph and Twitter Card meta tags for link previews
- Accessibility considerations: `prefers-reduced-motion` support, visible focus states
